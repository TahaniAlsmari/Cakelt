# Structural Modeling (Class Diagram)

##  System Architecture
The Class Diagram illustrates the static structure of the **Cakelt** system, showing the attributes and methods of each entity and their relationships.

![Class Diagram](../assets/ui-design.png)

##  Core Entities
* **User:** Stores credentials and profile data.
* **Cake:** Contains attributes for size, flavor, shape, and price calculation.
* **Order:** Links the User to their selected Cakes and manages the total amount.
* **Payment:** Handles transaction details and status.
